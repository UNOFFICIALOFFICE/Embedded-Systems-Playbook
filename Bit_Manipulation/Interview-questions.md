# 🔥 Bit Manipulation Interview Questions

A curated list of **must-know bit manipulation problems** for embedded systems and C programming interviews.

---

## 🧩 Basics

1. Check if a number is even or odd using a bitwise operator  
2. Swap two numbers without using a temporary variable  
3. Set, clear, toggle, and check a bit at position `n`  
4. Multiply or divide a number by 2 using bitwise operators  

---

## ⚡ Intermediate

5. Check if a number is a power of 2  
6. Count number of set bits (Brian Kernighan’s algorithm)  
7. Find the only non-repeating element in an array (XOR trick)  
8. Find two non-repeating elements in an array  
9. Reverse bits of a number  
10. Check if the `i-th` bit is set  

---

## 🚀 Advanced / Interview Favorites

11. Find missing number using XOR  
12. Find XOR of all numbers from `1` to `n` (pattern-based)  
13. Detect if two integers have opposite signs  
14. Turn off the rightmost set bit  
15. Isolate the rightmost set bit  
16. Count trailing zeros in binary representation  
17. Find position of only set bit  
18. Add two numbers without using `+` operator  
19. Divide two integers without using `/` operator  
20. Find subset using bit masking  

---

## 🧠 System-Level (Embedded Focus)

21. Convert **Big Endian ↔ Little Endian** (32-bit integer)  

---

## 🎯 How to Use This

- Try solving each problem **without looking at solutions**
- Focus on **bit-level reasoning**, not memorization
- Practice writing clean and efficient C code
- Be ready to explain your approach in interviews

---

## 💡 Tip

> In embedded systems, bit manipulation is not optional —  
> it's how you interact with hardware.
